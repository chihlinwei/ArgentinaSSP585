# Installation

### To install the package, open R and type:


``` r
install.packages("devtools")
```

### Then, you can install ArgentinaSSP585:


``` r
devtools::install_github("chihlinwei/ArgentinaSSP585", dependencies = TRUE)
library(ArgentinaSSP585)
```
